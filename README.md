# MTG-Combo-Optimizer---LP-BF-UB
A decision-support tool for storm combo sequencing using Magic the Gathering cards: Lotus Petal, Brain Freeze, and Underworld Breach.

These three cards create a well-known combo to mill either mill out all your opponents or to mill yourself out and win with Thassa's Oracle or Laboratory Maniac.
This application runs through the all the mathematics with a best-path greedy algorithm to identify the optimal sequence of card plays to win by milling out each opponent.
